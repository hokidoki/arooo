## install

```
$: yarn
```

## script

test : "jest"를 실행해서, 비즈니스로직을 검사합니다.
dev : 개발환경에서 실행합니다.
build : next앱을 구동시킬 파일을 빌드합니다.
start : "build"명령어로 만들어진 앱을 실행합니다.
